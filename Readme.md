# Streakly Roadmap & Milestones

Streakly
A lightweight habit-tracker focused on maintaining daily streaks, quick logging, and clear analytics. Mobile-first, offline-capable, and privacy-conscious.

### Repo Links

- Frontend: [streakly-fe — GitHub](https://github.com/Ananta580/streakly-fe) — React Router + Vite client
- Backend: [streakly-be — GitHub](https://github.com/Ananta580/streakly-be) — Express/Node API and Prisma
